# WC26 世界杯投注账本

> 朋友间世界杯娱乐，说一句话就记上。

通过对话管理 2026 世界杯投注。用户说什么，Agent 自动执行。

## 首次使用

用户第一次提到世界杯/投注时，执行初始化：

```bash
cd <skill目录> && npx tsx src/cli.ts init
```

然后告诉用户："世界杯账本已初始化，104 场比赛已加载。可以开始下注了。"

## 用户说什么 → Agent 做什么

### 下注

| 用户说 | Agent 行为 |
|:-------|:-----------|
| 韩国赢捷克 100 | 精确匹配韩国vs捷克 → 从API查赔率 → 下注 |
| 巴西赢 200 | 模糊匹配巴西的比赛 → 多场则列出候选 → 下注 |
| 记一笔韩国赢 50 | 同上 |
| 下注 | 显示所有可下注比赛 → 用户选择 → 下注 |
| 下注 100 | 同上，金额已知 |

**赔率从 API 实时获取，用户不需要输入。**

### 更新比分

| 用户说 | Agent 行为 |
|:-------|:-----------|
| 韩国 2-1 捷克 | 更新比分 → 自动结算该比赛所有注单 |
| 韩国 2 比 1 捷克 | 同上 |
| 巴西赢了 3-0 | 同上 |

### 查看信息

| 用户说 | Agent 行为 |
|:-------|:-----------|
| 我赢了多少 | 显示盈亏统计 |
| 看看账本 | 同上 |
| 看看 A 组 | 显示小组积分榜 |
| 世界杯积分榜 | 显示所有小组 |
| 今天的热搜 | 获取微博热搜 |

### 管理

| 用户说 | Agent 行为 |
|:-------|:-----------|
| 删掉韩国那注 | 删除指定注单 |
| 我要重置 | 重置所有数据（需确认） |

## Agent 执行规则

1. **赔率自动获取**：下注时从投注平台 API 获取实时赔率，用户不需要输入
2. **比赛自动匹配**：根据用户说的队伍名自动查找比赛 ID
3. **比分自动结算**：更新比分后自动结算该比赛的所有注单
4. **积分榜自动刷新**：超过 30 分钟自动从 API 刷新

## 定时结算

其他智能体可定时运行：

```bash
npx tsx src/cli.ts check-and-notify
```

输出 `data/notify.json`，有结算时才输出。
