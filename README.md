## 你什么时候需要它？

1. **朋友群娱乐**——"记一笔：韩国 vs 捷克，韩国赢，赔率 2.5，下注 100"
2. **比分更新自动算账**——"韩国 2 - 1 捷克" → 自动结算所有相关注单
3. **看盈亏**——"看看账本" → 净收益、收益率、胜率一目了然

## 它会交付什么？

- 注单管理（5 种玩法：胜平负/让球/大小球/波胆/自定义）
- 比分更新后自动结算
- 盈亏统计 + 收益率
- 积分榜（自动从 API 刷新）
- 淘汰赛对阵 + 预测
- 通知文件 `data/notify.json`（供其他 Agent 读取发消息）
- 实时赔率查询
- 数据分析报告（按天/按玩法/连胜连败）
- 社交媒体情绪分析
- AI 投注推荐（凯利公式 + 情绪分析）
- 连胜/连败检测和通知
- 多用户支持（独立盈亏统计）
- 导出报表（CSV/JSON）
- ELO 排名系统（32支球队）
- 历史数据回测（英超真实赔率 + 世界杯ELO估算）

## 回测结果

使用 football-data.co.uk 英超真实数据（2021-2024，1140场比赛）：

| 策略 | 说明 | 胜率 | 收益率 |
|:-----|:-----|:-----|:-----|
| 价值投注 | 寻找赔率与概率不匹配的选项 | 19.7% | **+21.5%** |
| 低赔率热门 | 选择赔率最低的选项 | 19.3% | -72.5% |
| 冷门策略 | 寻找势均力敌的比赛 | 21.9% | -78.1% |

⚠️ 世界杯数据使用 ELO 排名估算赔率（免费历史赔率数据无法获取）。

## 安全边界

- 不会联网调用外部 API（积分榜自动刷新有 30 分钟缓存，失败时静默回退）
- 不会删除用户数据（`reset` 命令需要 `--confirm true`）
- 不会泄露任何信息（纯本地 JSON 存储）
